# californication

RHCP Californication album chord and lyric guide. Browse songs and sections with chords and hints. Screen-only display, no grid required.

## Description

Californication is a study tool for Red Hot Chili Peppers fans and musicians. Browse every song from the Californication album with detailed harmonic analysis, chord progressions, and lyric hints. Perfect for learning the album by ear, understanding the harmonic structures, or preparing covers.

A screen-only norns application with no grid or external hardware required.

## Installation

`;install https://github.com/jamminstein/californication`

## Controls

- **E1**: Scroll through songs
- **E2**: Scroll through song sections
- **K2**: Previous song
- **K3**: Next song

## Features

- All 15 songs from Californication album
- Per-section chord analysis
- Lyric hints and markers
- Simple scroll-based navigation
- Detailed harmonic information
- Clean, readable display

## Requirements

- norns only
- No grid required
- No external engines needed
